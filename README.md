# Traffic Light Controller using ATmega32A

## Overview

This project implements a four-way traffic light control system using the ATmega32A microcontroller. The system controls traffic signals for North, South, East, and West directions using predefined timing sequences.

## Features

- Four-way traffic signal control
- Timing-based signal sequencing
- Embedded C implementation
- Proteus simulation

## Tools Used

- ATmega32A
- Embedded C
- Proteus Design Suite

## Repository Structure

```text
src/       -> Source code
images/    -> Simulation screenshots
```

## Simulation

The project was simulated in Proteus to verify traffic light sequencing and timing behavior.
